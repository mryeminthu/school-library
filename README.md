# 📖 School Library<a name="about-project"></a>

- School Library was developed in Ruby to assist librarians in efficiently managing their tasks by saving records of book loans, including borrower information and loan dates.

## 🛠 Built With <a name="built-with"></a>

  </ul>
</details>

<!-- Features -->

### Key Features <a name="key-features"></a>

- **Add new students**
- **Add new teachers**
- **Add new books**
- **Keep track of who borrowed a specific book and when they did it**
